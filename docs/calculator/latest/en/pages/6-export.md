# Export

Exporting your input files, calculation job or PDF report is the only way to 'save' your work from this service. All files are exported as .zip files that can be reimported directly from the home page without extracting the files. Exported files can also be viewed in external tools, such as QGIS.

There are three options for exporting your work:
- **Input files** - export a Geography Markup Language (GML) file containing all the information entered in the 'Input' section, such as emission sources and buildings. This does not include calculation job settings or results. You can use this option to export scenarios you are not finished entering to continue working on another time.
- **Calculation job** - export a GML file containing all inputs, calculation settings and results associated with a single calculation job, including met data details. Use the 'Choose calculation job' dropdown to specify which calculation job you would like to export. You do not need to calculate the calculation job before exporting it. You can use this option to export entire calculation jobs to reimport another time to view the results.
- **Report** - export a report in PDF format for either process contribution or in-combination results, using the 'Report type' dropdown. The report will also contain the GML information and can be reimported to view the full results. You can use this option to create a file to be submitted as part of an air quality impact assessment in a planning or permitting application.

When exporting a file, you must provide your email address. This is to send a link to download your file when it is ready and will not be kept for anything else. Additionally, if an error occurs during the calculation and the application is closed an email will be sent to the email address you provide.

You can also download files directly by using the download link shown in the message centre when the files are ready.

For calculations that are expected to take a while to run, you can exit the service after starting the export. You will be notified by email when the calculation job is complete and the results can be downloaded.
